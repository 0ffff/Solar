通信功能
用于通过以太网接口与MX100通信的功能。

设置功能
用于配置MX100，模块和通道。但是，CAN总线模块
硬件设置不受支持。对于硬件设置，请使用该软件
随模块一起提供。

数据检索功能
•使用FIFO检索测量数据
通过FIFO（先进先出）缓冲区检索最近测量的数据。
也检索瞬时值。瞬时值的检索间隔
是100毫秒或更多。
•按通道检索测量数据
检索指定通道的最新测量数据。也检索
瞬时值。瞬时值的检索间隔为100 ms或
更多。
•检索设置数据
检索MX100的系统配置，网络信息如
IP地址和与频道相关的设置数据。
•集体检索DO（数字输出）数据
集体检索DO的ON / OFF状态。
•检索频道信息数据（频道号码等）
•检索AO / PWM数据
检索显示AO / PWM通道输出的数据。
•检索传输输出数据
表示AO / PWM通道传输状态的数据。
当数据被检索时，它指示当前传输输出的状态。
数据发送时，它是控制该数据的开始和停止的规范
传输输出。

控制功能
•日期/时间设置
•设置将测量数据保存到CF的备份功能
卡和格式化CF卡
•系统的初始化和重新配置（单位）
•重置警报（确认警报）
•指定7段LED上显示的信息

公用事业
提供将测量值转换为字符串的功能，用于检索
错误消息字符串和其他实用程序功能。
